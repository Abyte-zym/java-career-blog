☕ Java面试一页纸速查表 2026.05

HashMap

结构：数组+链表+红黑树(>8转树,<6退链)
put：hash→index→空则放/有则链/树则putTreeVal
扩容：size>threshold(容量×0.75)→2倍→rehash
1.7vs1.8：头插→尾插(防死循环)

ConcurrentHashMap

1.7：Segment分段锁(ReentrantLock)
1.8：CAS+synchronized锁头节点
扩容：多线程并发迁移(sizeCtl)

JVM

内存：堆/栈/方法区(元空间)/程序计数器/本地方法栈
GC算法：标记清除/复制/标记整理/分代
CMS：初始标记→并发标记→重新标记→并发清除
G1：Region分区·可预测停顿·Mixed GC
类加载：双亲委派·Bootstrap→Extension→App→Custom

并发

synchronized：偏向→轻量(自旋)→重量(OS互斥量)
volatile：可见性+禁重排·不保证原子性
AQS：CLH队列+state+CAS·ReentrantLock/Semaphore基类
线程池：corePool→queue→maxPool→reject

设计模式

单例DCL：volatile+双重检查+synchronized
Spring用了：工厂/代理/模板方法/观察者/策略/适配器/装饰器/单例/原型/责任链

Spring Boot

@SpringBootApplication=@Configuration+@EnableAutoConfiguration+@ComponentScan
自动配置：META-INF/spring.factories→条件装配
IoC容器：BeanFactory(基础)·ApplicationContext(高级)
Bean生命周期：实例化→属性注入→Aware→BPP前置→init→BPP后置→使用→destroy

AOP & 事务

动态代理：JDK(接口)·CGLIB(继承)
@Transactional失效：①非public ②自调用 ③异常被catch ④rollbackFor不对 ⑤final类

MySQL

B+树：数据存叶子·叶子双向链表·非叶子只存key
最左前缀：(a,b,c)→a可用，a+b可用，只有b不可用
索引失效：①LIKE '%x' ②OR非索引 ③函数操作 ④隐式转换 ⑤!=/<> ⑥IS NULL
EXPLAIN重点：type(ALL

Redis

5类型：String·Hash·List·Set·ZSet
穿透：布隆过滤器+缓存空值
击穿：互斥锁+逻辑过期+热点永不过期
雪崩：随机TTL+多级缓存+限流
持久化：RDB(快照)+AOF(追加命令)

消息队列

RocketMQ：NameServer+Broker·事务消息(半消息+回查)
Kafka：高吞吐·分区有序·ISR机制
不丢：同步刷盘+手动ACK+重试

分布式

CAP：一致性·可用性·分区容忍→只能同时满足两个
分布式事务：2PC·TCC·消息表·Seata AT
分布式锁：Redis SETNX+过期+WatchDog自动续期
雪花算法：1位+41位时间+10位机器+12位序列

Spring Cloud Alibaba

Nacos：服务注册+配置中心(AP模式)
Sentinel：流量控制·熔断降级·系统保护
Gateway：路由·鉴权·限流·跨域
Seata：AT模式(自动回滚)·TCC模式

系统设计

短URL：发号器(Base62)+缓存+302重定向
秒杀：Redis Lua扣库存+RocketMQ削峰+Sentinel限流

AI（2026必问）

Agent=LLM+Planning+Memory+Tools
RAG：文档分块→向量检索→Prompt增强→生成回答
Spring AI：ChatClient+EmbeddingModel+VectorStore

计算机网络

TCP三次握手：SYN→SYN+ACK→ACK
HTTPS：非对称加密交换对称密钥→对称加密通信

Docker

镜像分层：每层只读·Copy-on-Write
Dockerfile：FROM→RUN→COPY→EXPOSE→CMD
