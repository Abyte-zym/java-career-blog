1. LRU缓存机制（频次：⭐⭐⭐⭐⭐）

实现LRU(最近最少使用)缓存，支持get和put操作，O(1)时间复杂度。
class LRUCache {
    class Node { int key, val; Node prev, next; }
    Map<Integer, Node> map = new HashMap<>();
    Node head, tail; int capacity;
    
    void remove(Node n) { n.prev.next = n.next; n.next.prev = n.prev; }
    void addToHead(Node n) { n.next = head.next; n.prev = head; head.next.prev = n; head.next = n; }
    
    int get(int key) {
        if (!map.containsKey(key)) return -1;
        Node n = map.get(key);
        remove(n); addToHead(n);
        return n.val;
    }
    
    void put(int key, int val) {
        if (map.containsKey(key)) { remove(map.get(key)); }
        Node n = new Node(); n.key = key; n.val = val;
        addToHead(n); map.put(key, n);
        if (map.size() > capacity) {
            Node last = tail.prev; remove(last); map.remove(last.key);
        }
    }
}

关键：HashMap+双向链表。get时移到头部，put超容量时删尾部。

2. 二叉树层序遍历（频次：⭐⭐⭐⭐⭐）

给定二叉树，返回其层序遍历结果（逐层从左到右）。
List<List<Integer>> levelOrder(TreeNode root) {
    List<List<Integer>> res = new ArrayList<>();
    if (root == null) return res;
    Queue<TreeNode> q = new LinkedList<>();
    q.offer(root);
    while (!q.isEmpty()) {
        int size = q.size();
        List<Integer> level = new ArrayList<>();
        for (int i = 0; i < size; i++) {
            TreeNode node = q.poll();
            level.add(node.val);
            if (node.left != null) q.offer(node.left);
            if (node.right != null) q.offer(node.right);
        }
        res.add(level);
    }
    return res;
}

BFS标准模板。每层size记录当前层节点数。时间O(n)，空间O(n)。

3. 快排及其优化（频次：⭐⭐⭐⭐）

手写快速排序，并说明如何避免最坏O(n²)。
void quickSort(int[] arr, int l, int r) {
    if (l >= r) return;
    int pivot = partition(arr, l, r);
    quickSort(arr, l, pivot - 1);
    quickSort(arr, pivot + 1, r);
}

int partition(int[] arr, int l, int r) {
    // 随机选pivot，避免最坏情况
    int randIdx = l + new Random().nextInt(r - l + 1);
    swap(arr, randIdx, r); // 移到最右
    int pivot = arr[r], i = l - 1;
    for (int j = l; j < r; j++)
        if (arr[j] <= pivot) swap(arr, ++i, j);
    swap(arr, i + 1, r);
    return i + 1;
}

平均O(n log n)。随机化pivot + 三数取中避免最坏。数据量小时切插入排序（<20个元素）。

4. 最长无重复子串（频次：⭐⭐⭐⭐）

给定字符串s，找出不含重复字符的最长子串长度。滑动窗口。
int lengthOfLongestSubstring(String s) {
    int[] lastIdx = new int[128]; // ASCII
    Arrays.fill(lastIdx, -1);
    int maxLen = 0, left = 0;
    for (int right = 0; right < s.length(); right++) {
        char c = s.charAt(right);
        if (lastIdx[c] >= left) left = lastIdx[c] + 1;
        lastIdx[c] = right;
        maxLen = Math.max(maxLen, right - left + 1);
    }
    return maxLen;
}

滑动窗口经典。left收缩条件是重复字符位置≥当前窗口左边界。时间O(n)。

5. 两数之和 & 三数之和（频次：⭐⭐⭐⭐）

两数之和用HashMap；三数之和用排序+双指针，去重是核心难点。
// 两数之和 O(n)
int[] twoSum(int[] nums, int target) {
    Map<Integer, Integer> map = new HashMap<>();
    for (int i = 0; i < nums.length; i++) {
        int need = target - nums[i];
        if (map.containsKey(need)) return new int[]{map.get(need), i};
        map.put(nums[i], i);
    }
    return new int[]{};
}

// 三数之和 O(n²) 
List<List<Integer>> threeSum(int[] nums) {
    Arrays.sort(nums);
    List<List<Integer>> res = new ArrayList<>();
    for (int i = 0; i < nums.length - 2; i++) {
        if (i > 0 && nums[i] == nums[i-1]) continue; // 去重
        int l = i + 1, r = nums.length - 1;
        while (l < r) {
            int sum = nums[i] + nums[l] + nums[r];
            if (sum == 0) {
                res.add(List.of(nums[i], nums[l], nums[r]));
                while (l < r && nums[l] == nums[l+1]) l++; // 去重
                while (l < r && nums[r] == nums[r-1]) r--;
                l++; r--;
            } else if (sum < 0) l++; else r--;
        }
    }
    return res;
}

6. 反转链表（频次：⭐⭐⭐⭐）

原地反转单链表。迭代和递归两种写法都要会。
// 迭代 O(n) O(1)
ListNode reverseList(ListNode head) {
    ListNode prev = null, curr = head;
    while (curr != null) {
        ListNode next = curr.next;
        curr.next = prev;
        prev = curr; curr = next;
    }
    return prev;
}

// 递归 O(n) O(n)
ListNode reverseRecursive(ListNode head) {
    if (head == null || head.next == null) return head;
    ListNode newHead = reverseRecursive(head.next);
    head.next.next = head; // 4→5→4 变成 4←5
    head.next = null;
    return newHead;
}

7. 合并K个有序链表（频次：⭐⭐⭐）

N个有序链表合并为一个。优先队列解法 + 分治解法。
// 优先队列 O(N log K)
ListNode mergeKLists(ListNode[] lists) {
    PriorityQueue<ListNode> pq = new PriorityQueue<>((a,b)->a.val-b.val);
    for (ListNode node : lists) if (node != null) pq.offer(node);
    ListNode dummy = new ListNode(0), tail = dummy;
    while (!pq.isEmpty()) {
        ListNode min = pq.poll();
        tail.next = min; tail = min;
        if (min.next != null) pq.offer(min.next);
    }
    return dummy.next;
}

面试官追问：时间复杂度？→ N个节点、K个链表，每次poll O(log K)，总O(N log K)。

8. 二叉树最大深度 & 判断平衡（频次：⭐⭐⭐）

求二叉树最大深度；判断是否为平衡二叉树（左右子树高度差≤1）。
// 最大深度
int maxDepth(TreeNode root) {
    return root == null ? 0 : Math.max(maxDepth(root.left), maxDepth(root.right)) + 1;
}

// 平衡二叉树 O(n) 自底向上
boolean isBalanced(TreeNode root) {
    return checkHeight(root) != -1;
}
int checkHeight(TreeNode node) {
    if (node == null) return 0;
    int left = checkHeight(node.left);
    int right = checkHeight(node.right);
    if (left == -1 || right == -1 || Math.abs(left-right) > 1) return -1;
    return Math.max(left, right) + 1;
}

-1标记法：既返回高度又传递不平衡信号，避免重复计算。

10. 动态规划 — 背包问题（频次：⭐⭐⭐）

0-1背包：容量W，n个物品(重量w[i],价值v[i])，最大价值。二维→一维优化。
// 二维 O(n*W)
int knapsack(int W, int[] w, int[] v) {
    int n = w.length;
    int[][] dp = new int[n+1][W+1];
    for (int i=1; i<=n; i++) {
        for (int j=0; j<=W; j++) {
            dp[i][j] = dp[i-1][j];
            if (j >= w[i-1])
                dp[i][j] = Math.max(dp[i][j], dp[i-1][j-w[i-1]] + v[i-1]);
        }
    }
    return dp[n][W];
}

// 一维优化 O(n*W) O(W)
int knapsack1D(int W, int[] w, int[] v) {
    int[] dp = new int[W+1];
    for (int i=0; i=w[i]; j--)  // 倒序！防止重复使用
            dp[j] = Math.max(dp[j], dp[j-w[i]] + v[i]);
    return dp[W];
}

一维倒序递推是面试难点——正序会变成完全背包（物品无限用）。
