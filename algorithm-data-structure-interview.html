Q1: 手写快速排序，并分析时间复杂度

面试话术："快排核心是分治——选一个pivot，把小于它的放左边，大于它的放右边，递归处理。平均O(nlogn)，最坏O(n²)出现在已排序数组每次选最小/最大元素时。随机化pivot可以避免最坏情况。"

public void quickSort(int[] arr, int left, int right) { if (left >= right) return; int pivot = partition(arr, left, right); quickSort(arr, left, pivot - 1); quickSort(arr, pivot + 1, right); } int partition(int[] arr, int l, int r) { int pivot = arr[r], i = l - 1; for (int j = l; j < r; j++) if (arr[j] <= pivot) { i++; swap(arr, i, j); } swap(arr, i + 1, r); return i + 1; }
Q2: 手写LRU缓存（Least Recently Used）

面试话术："LRU用HashMap+双向链表实现。HashMap保证O(1)查找，双向链表维护访问顺序。每次get把节点移到链表头，put时如果满了就删链表尾节点。LinkedHashMap默认就是LRU——构造参数accessOrder=true即可。"

class LRUCache { int cap; LinkedHashMap<Integer, Integer> map; LRUCache(int c) { cap = c; map = new LinkedHashMap<>(c,0.75f,true) { protected boolean removeEldestEntry(Map.Entry e) { return size() > cap; }}; } int get(int k) { return map.getOrDefault(k,-1); } void put(int k, int v) { map.put(k,v); } }
Q3: 反转单链表（迭代+递归）

面试话术："迭代法用三个指针prev/curr/next逐个反转。递归法先反转子链表，再让当前节点的next指向prev。时间复杂度O(n)，空间O(1)迭代/O(n)递归栈。"

// 迭代 public ListNode reverse(ListNode head) { ListNode prev = null, curr = head; while (curr != null) { ListNode next = curr.next; curr.next = prev; prev = curr; curr = next; } return prev; }
Q4: 二叉树的前序/中序/后序遍历（非递归）

面试话术："递归本质是栈——手动维护一个Stack模拟。前序：根→左→右，入栈顺序右→左。中序：把所有左子节点入栈，弹出时访问，然后处理右子节点。"

public List<Integer> inorder(TreeNode root) { List<Integer> res = new ArrayList<>(); Stack<TreeNode> s = new Stack<>(); TreeNode cur = root; while (cur != null || !s.isEmpty()) { while (cur != null) { s.push(cur); cur = cur.left; } cur = s.pop(); res.add(cur.val); cur = cur.right; } return res; }
Q5: 最长不重复子串（滑动窗口）

面试话术："滑动窗口模板：右指针扩窗口，左指针缩窗口。HashMap记录字符最后出现的位置，遇到重复就把左指针跳到重复字符的下一个位置。O(n)时间，HashMap存字符索引。LeetCode 3原题。"

public int lengthOfLongestSubstring(String s) { Map<Character, Integer> map = new HashMap<>(); int max = 0, left = 0; for (int i = 0; i < s.length(); i++) { if (map.containsKey(s.charAt(i))) left = Math.max(left, map.get(s.charAt(i)) + 1); map.put(s.charAt(i), i); max = Math.max(max, i - left + 1); } return max; }
Q6: HashMap底层原理 & 为什么用红黑树

面试话术："Java 8的HashMap是数组+链表+红黑树。hash→取模定位桶，桶内<8个元素用链表，≥8且数组≥64时转红黑树。红黑树查找O(logn)远优于链表O(n)，解决哈希碰撞严重时的性能退化。扩容时rehash，容量翻倍，每个元素重新计算位置。"

Q8: 动态规划——爬楼梯/背包问题

面试话术："DP核心是状态定义+状态转移方程。爬楼梯：dp[i]=dp[i-1]+dp[i-2]，本质是斐波那契，空间可优化到O(1)。0-1背包：dp[i][w]=max(dp[i-1][w], dp[i-1][w-wt[i]]+val[i])，可压缩为一维逆序遍历。"

// 爬楼梯 public int climbStairs(int n) { if (n <= 2) return n; int a=1, b=2; for (int i=3; i<=n; i++) { int t=a+b; a=b; b=t; } return b; }
Q9: 堆排序 & TopK问题

面试话术："TopK用小顶堆：维护大小为K的最小堆，遍历数组，比堆顶大就替换。Java用PriorityQueue默认就是小顶堆。时间复杂度O(nlogK)，空间O(K)。全量排序用堆排序：建堆O(n)→依次弹出O(nlogn)，不稳定排序。"

// TopK public int findKthLargest(int[] nums, int k) { PriorityQueue<Integer> pq = new PriorityQueue<>(); for (int n : nums) { pq.offer(n); if (pq.size() > k) pq.poll(); } return pq.peek(); }
Q10: 图的BFS/DFS——岛屿数量/最短路径

面试话术："图遍历分BFS和DFS。岛屿数量用DFS：遍历grid，遇'1'就DFS把整座岛沉为'0'，count++。最短路径用BFS：逐层扩散，第一次到达终点即最短。DFS用栈/递归，BFS用队列。"
