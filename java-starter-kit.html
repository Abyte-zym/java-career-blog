🏗️ 项目结构

java-starter-kit/
├── src/main/java/com/starter/
│   ├── common/
│   │   ├── R.java              ← 统一响应体
│   │   ├── BusinessException   ← 业务异常
│   │   └── GlobalExceptionHandler ← 全局捕获
│   ├── config/
│   │   ├── JwtConfig.java      ← JWT配置
│   │   ├── SecurityConfig      ← Spring Security
│   │   └── SwaggerConfig       ← 接口文档
│   ├── controller/             ← 示例接口
│   └── StarterApplication.java ← 启动类
├── docker-compose.yml          ← MySQL+Redis+App
├── Dockerfile                  ← 多阶段构建
└── application.yml             ← 即插即用
